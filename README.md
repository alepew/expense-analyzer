Привет! Это моя первая программа для учета личных финансов. 

# Возможности
- Сбор данных о тратах
- Расчет общей суммы и среднего чека
- Поиск самых крупных и мелких трат
- Сохранение отчета в файл

# Как запустить?
1. Установи Python
2. Скачай 'expense_analyzer.py'
3. Запусти: 'python expense_analyzer.py'

# Чему я научилась?
- Работа с функциями 
- Обработка ошибок
- Работа со списками и словарями
- Сохранение данных в файл
